# ConnectaTel User Behavior Analysis – Sprint 7 Final

Este repositorio contiene el análisis del comportamiento de usuarios de ConnectaTel, enfocado en la limpieza de datos, análisis exploratorio (EDA), segmentación de clientes y generación de insights para la toma de decisiones.

El dataset incluye información de uso (llamadas, mensajes y duración), con presencia de valores faltantes, sentinels y outliers, simulando un escenario real de análisis.

## 🧠 Objetivo del análisis
Identificar problemas de calidad de datos
Limpiar y preparar el dataset
Analizar patrones de uso de los usuarios
Detectar outliers relevantes
Segmentar clientes por uso y edad
Generar insights para el negocio
## 📊 Principales hallazgos
La mayoría de los usuarios se concentra en el segmento de uso medio
Existe un grupo reducido de alto uso, clave para ingresos
Se detectaron outliers en minutos de llamadas (usuarios intensivos)
Predominan los adultos y adultos mayores
## 💡 Impacto para el negocio
Identificación de clientes de alto valor
Oportunidades de crecimiento en usuarios de bajo uso
Base para diseñar planes diferenciados
Mejora en estrategias de segmentación
